<p>1. Clone a Github Repository:</p>

```

```

<p>2. From cloned folder navigate to source folder:</p>

```

```

<p>3. Install dependencies:</p>

```

```

<p>4. Run project with a dev server:</p>

```
npm start
```

<p>5. Open:</p>

```
http://localhost:4200/
```
